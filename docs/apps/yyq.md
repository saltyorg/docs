# yq

## Overview

yq is a lightweight, portable command-line tool designed to process YAML, JSON, XML, INI, properties, CSV, and TSV files using a syntax similar to jq, which is used for JSON processing.

---

## Deployment

```sh
sb install yyq
```

## Usage

```sh
yyq
```
