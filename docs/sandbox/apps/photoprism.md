### 1. Installation

``` shell

sb install sandbox-photoprism

```

### 2. URL

- To access Photoprism, visit `https://photoprism._yourdomain.com_`

### 3. Setup

- Default login:

  ``` { .yaml}
  Username: admin
  Password: your_normal_password
  ```
