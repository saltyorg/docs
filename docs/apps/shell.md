# Shell

## Overview

Saltbox dependency.

Installs and configures a user's shell (Bash or Zsh) with tools like z for directory jumping, argcomplete for tab completion, Oh My Zsh and optional Oh My Posh theming.

---

## Deployment

```sh
sb install shell
```
