# Bazarr

## What is it?

[Bazarr](https://www.bazarr.media/) is a companion application to Sonarr and Radarr that manages and downloads subtitles based on your requirements.

### 1. Installation

``` shell

sb install bazarr

```

### 2. URL

- To access Bazarr, visit `https://bazarr._yourdomain.com_`
