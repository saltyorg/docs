# Saltbox Aliases

Handy guide to Saltbox Command line aliases.
## sb update

- Update Saltbox

- Usage:

    ``` { .shell }

        sb update

    ```

## sb list

- List Saltbox packages

- Usage:

    ``` { .shell }

        sb list

    ```

## sb install `<package>`

- Install `<package>`

- Usage:

    ``` { .shell }

        sb install plex

    ```

## sb install cm-`<package>`

- Install community `<package>`

- Usage:

    ``` { .shell }

        sb install cm-jellyfin

    ```
